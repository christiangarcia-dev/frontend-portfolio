Projects

Mindstill

An immersive installation designed to lead viewers on a visual journey reflecting the vivid imagery that emerges during meditation. This project merges video editing and procedural visual programming in an effective manner, utilizing TouchDesigner and Adobe Premiere Pro. Other elements include projection and spatial audio. Used TouchDesigner to create designs displayed in the final video sequence. Used Premiere Pro to stitch together individual clips.

GymPass

Built an Android app that acts as an intermediary that connects its users with athletic facilities for temporary access. Used Google Places API, Firebase Realtime Database, and Firebase Authentication to fetch and write user/place data into the database. Implemented layout designs with XML.

Unity Space Game

Built a three dimensional space exploration game using my own Blender models as well as imported ones. The objective of the game is to allow the user to travel through outer space and visualize planets, satellites, and cosmic phenomena.

Portfolio Website

Designed my personal website using hand-drawn sketches and Figma. Proceeded to build it using raw HTML, CSS, and JS. No frameworks were used throughout the development of this website.
